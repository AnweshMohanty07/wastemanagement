# EcoPulse - Waste Management System Website

A responsive, interactive website for a Waste Management System, built using HTML, CSS, and JavaScript with a 3D animated hero scene powered by Three.js.

## Features

- Modern responsive layout for desktop, tablet, and mobile
- Smooth scrolling and section reveal animations
- Interactive waste reduction calculator
- Animated impact counters
- Contact form UI (frontend)
- Theme shift mode toggle
- Real-time 3D visual animation using Three.js

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Three.js (CDN)

## Getting Started

1. Clone or download this repository.
2. Open the project folder.
3. Run the site by opening `index.html` in a browser.

For better development workflow, use a local server (for example VS Code Live Server).

## Usage

- Use top navigation to move between sections.
- Use sliders in the calculator to estimate monthly landfill reduction.
- Click `Shift Look` to change visual theme.
- View the 3D animated scene in the hero section.
